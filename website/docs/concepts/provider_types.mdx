## Different Types of Providers

With all the following provider types available, when should you use which provider?
The answer depends on your use case and what you are most comfortable with.

| Provider Type          | Description                                                                                   |
| ---------------------- | --------------------------------------------------------------------------------------------- |
| Provider               | Provides for plain dart objects                                                               |
| FutureProvider         | Provider for async values                                                                     |
| StreamProvider         | Provider for streams                                                                          |
| StateProvider          | Provider for a value that can be changed externally                                           |
| StateNotifierProvider  | Provider for StateNotifiers allowing complex immutable objects with an interface for update   |
| ChangeNotifierProvider | Provider for ChangeNotifiers allowing for complex mutable state with an interface for changes |
